Javascript

Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

Variable

JavaScript variables are containers for storing data values.

Variable Naming Conventions

Variable and function names are written as camelCase with descriptive names

Functions

A JavaScript function is a block of code designed to perform a particular task and is executed when "something" invokes it (calls it).

Methods

JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

Function arguments are the real values passed to (and received by) the function.

Parameters

Function parameters are the names listed in the function definition.

Return

The return statement stops the execution of a function and returns a value from that function.

Chaining Methods

Repeatedly calling one method after another on an object, in one continuous line of code.

Strings

A JavaScript string stores a series of characters

Booleans

A JavaScript Boolean represents one of two values: true or false.

Undefined

The undefined property indicates that a variable has not been assigned a value, or not declared at all.

NaN

NaN, standing for not a number, is a numeric data type value representing an undefined or unrepresentable value

Escape

The backslash (\) escape character turns special characters into string characters

Alert

The alert() method displays an alert box with a specified message and an OK button.

Comments

When you place these two slashes, all text to the right of them will be ignored, until the next line.

jQuery

jQuery is a concise and fast JavaScript library that can be used to simplify event handling, HTML document traversing, Ajax interactions and animation for speedy website development.

Attributes

Attributes provide additional information about HTML elements.
